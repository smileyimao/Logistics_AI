用了这个工具之后：**图片发出去，Excel自动生成，人工录入这步直接省掉。**

## 怎么开始用（就这一步）

扫码关注公众号，完成。

不用下载任何APP，不用安装任何软件，手机电脑都能用。

## 日常怎么用

### 员工的操作

跟平时一样在群里发图，只多一个动作：

```
把图片转发给这个公众号
```

发完之后，公众号自动回复：

> ✅ 已录入：SF123456789

表示这张单已经识别并保存好了。

就这一步，其他什么都不用管。

---

### 老板的操作

当天处理完了，想拿数据，发一条消息：

| 发送 | 得到 |
|------|------|
| `导出` | 今天所有运单的Excel |
| `导出本月` | 本月全部运单 |
| `导出上月` | 上月全部运单 |
| `导出 2026-04` | 指定任意月份 |
| `导出全部` | 从开始到现在所有数据 |

公众号回复一个下载链接，点开直接下载Excel文件，打开就是整理好的表格。

## 权限分工

| 角色 | 能做的事 |
|------|---------|
| 老板 | 发图录入 + 导出Excel + 管理员工 |
| 员工 | 只能发图录入，不能导出 |

老板注册后拿到一个6位邀请码，发给员工，员工发一条「加入 XXXXXX」就绑定了。

---

## 识别效果

AI直接从图片中读取以下字段，自动填入Excel：

- 运单号
- 发货人 / 收货人
- 收货地址
- 重量 / 件数
- 货物品名
- 备注

图片清晰的情况下准确率95%以上。识别有误的字段可以在Excel里手动改。
